## Development Guide

1. Clone the project.

    ```sh
    git clone git@github.com:sdescarries/pwgen.git
    cd pwgen
    ```

2. Install dependencies.

    ```sh
    yarn
    ```

3. Serve with hot reload at <http://localhost:3000/> .

    ```sh
    yarn start
    ```
